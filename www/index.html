Stats tracking

Death counts

!death
Adds 1 death to counter for the current game
!death add #
(mod only) Adds # deaths to counter for the current game
!death remove #
(mod only) Removes # deaths from counter for the current game
!deathCount
Post total deaths for the current game
!totalDeath
Post total deaths for every game

Diamond counts (Minecraft)

!diamond
Adds 1 diamond to counter for the current game
!diamond add #
(mod only) Adds # diamonds to counter for the current game
!diamond remove #
(mod only) Removes # diamonds from counter for the current game
!diamondCount
Post total diamonds for the current game

Flunge counts (Magic)

!flunge
Adds 1 flunge to counter for the current game
!flunge add #
(mod only) Adds # flunges to counter for the current game
!flunge remove #
(mod only) Removes # flunges from counter for the current game
!flungeCount
Post total flunges for the current game

Tournament record

!matchwin
Add a win to counter
!matchwin remove
(mod only) Remove a win from counter
!matchlose
Add a loss to counter
!matchlose remove
(mod only) Remove a loss from counter
!matchresult
Post current match results
!matchreset
(mod only) resets match counters
